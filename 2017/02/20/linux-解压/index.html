*.rar unrar e
*.zip unzip

一、gzip

1. 命令格式：gzip 文件名

已文件 file 为例进行压缩：

gzip file

此时 file 文件已经不存在，取而代之的是 file.gz 文件。

也就是说 gzip 不保留原文件，压缩完成之后，只剩一个压缩包，原文件已经消失不见。

2. 另外，gzip 只能压缩文件，不能压缩目录。

如想要压缩 test-dir 目录，命令执行失败：

gzip test-dir/
gzip: test-dir/ is a directory -- ignored

3. 解压命令：

gunzip filename.gz

或者

gzip -d filename.gz

二、tar

tar 可以用来压缩目录

压缩：

  • -c 打包
  • -v 显示详细信息
  • -f 指定文件名
  • -z 打包同时压缩

解压缩命令

  • -x 解包
  • -v 显示详细信息
  • -f 指定解压文件
  • -z 解压缩
