按钮添加一个类名自动变loading动画版

2018-12-13

Contents

有了下面的 css 代码，给任意的标签（button、a、span等等）加上一个类名 spin-loading，就能使标签自动变成 loading 状态，方便快捷效果好。

PS: 小程序中的 view 和 text 标签也同样适用。

/* 按钮loading */
.spin-loading {
  position: relative;
}

.spin-loading::before {
  /* 定位：水平垂直居中 */
  position: absolute;
  top: 0;
  right: 0;
  bottom: 0;
  left: 0;
  margin: auto;

  /* 基本样式 */
  width: 4px;
  height: 4px;
  content: '';
  border-radius: 100%;

  /* 动画 */
  animation: spinZoom 1s steps(8) infinite;
  /* 小圆点 */
  box-shadow:
    0 -10px 0 1px currentColor,
    10px 0 currentColor,
    0 10px currentColor,
    -10px 0 currentColor,
    -7px -7px 0 .5px currentColor,
    7px -7px 0 1.5px currentColor,
    7px 7px currentColor,
    -7px 7px currentColor;
}

@keyframes spinZoom {
  0% {
    transform: rotate(0);
  }

  100% {
    transform: rotate(360deg);
  }
}
