vue中父级组件通过Prop向自组件传递数据

2019-01-08

Contents

      • Props

Props

以按钮组件 <g-button> 的 disabled 属性为例：

<template>
  <button :disabled="disabled"></button>
</template>
<script>
  export default {
    props: {
      disabled: {
        type: Boolean,
        default: false
      },
    }
  }
</script>

在某父组件中使用该按钮组件：

1
2
3
4
5

<template>
  <div>
    <a-button disabled>Primary</a-button>
  </div>
</template>

按钮组件中定义了是否禁用 disabled 属性，它由父级传入，通过 props 选项接收。

props 选项可以是一个字符串数组，元素为属性名，如 ['disabled']，也可以是一个对象，推荐使用对象格式，因为这样可以对属性进行更多的处理：类型检查、必填校验、自定义校验和指定默认值。

例如这里的 disabled 属性，它的默认值就是 false，它的类型就是布尔值，如果在父组件中使用时传入一个字符串则会在控制台产生一个警告。

上面例子展示了如何在父组件中向子组件传递数据以及如何接收数据。关于 Prop 详细的使用可以参考官方文档
