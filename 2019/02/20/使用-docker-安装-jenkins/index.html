使用 docker 安装 Jenkins

Written by with ♥ on 2019-02-20 in Docker

使用 docker 安装 Jenkins：

docker run \
  -u root \
  --rm \
  -d \
  -p 8080:8080 \
  -p 50000:50000 \
  -v jenkins-data:/var/jenkins_home \
  -v /var/run/docker.sock:/var/run/docker.sock \
  jenkinsci/blueocean

安装成功之后进入设置向导（访问 域名:8080），需要输入密码来解锁 Jenkins。密码会在 Jenkins 控制台日志中打印出来，如果是使用 docker 安装的 Jenkins，可以通过 docker 日志来访问 Jenkins 的控制台日志。

简而言之就是通过查看 docker 日志得到密码。

docker log <container-id>
